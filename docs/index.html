Architecture details are private

The full Rozi X technical architecture document is not published on this site while intellectual-property review is underway.

Public materials stay at the capability and principle level:

  • Rozi does not treat an agent’s completion claim as proof.
  • Completion requires externally checkable evidence.
  • The system can reconcile claimed results against observed environment state.
  • Authority picture: Operator · Rozi · External — people stay distinct; see the Architecture Voyage.

The method, in one sentence: a single-LLM control architecture in which an agent’s completion assertion is non-authoritative; completion is gated by independently observable, task-specific environment predicates and preserved evidence receipts.
